## 📖 About education-auth

education-auth is a full-stack login system designed for schools. It lets different users—students, teachers, parents, and admins—sign in securely. It uses popular and trusted tools like Node.js, Express, MongoDB Atlas, JWT, and bcrypt. Each user gets access only to what they need. This helps keep information safe and organized.

## 🔒 What Happens Behind the Scenes (Simplified)

education-auth uses a few tools to make your login safe and secure.

- **bcrypt** protects your password by turning it into a secret code.
- **JWT (JSON Web Tokens)** lets the app remember you safely without passing your password every time.
- **MongoDB Atlas** stores user details on the cloud securely.
- **Express and Node.js** run the server that talks with your app.

You only need to use the app as shown. These tools work invisible to keep your data safe.
